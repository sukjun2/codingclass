프로그래머스 0단계 : 몫 구하기

정수 num1, num2가 매개변수로 주어질 때, num1을 num2로 나눈 몫을 return 하도록 solution 함수를 완성해주세요.


※ 제한사항

0 < num1 ≤ 100
0 < num2 ≤ 100

※ 입출력 예

num1 num2 result
10 5 2
7 2 3 

function solution(num1, num2) {
    let answer = 0;
    answer = Math.floor(num1 / num2);
    return answer;
}

num1을 num2로 나눈값을 수학 메서드 Math.floor()을 사용하여 소수점을 버린 값으로 계산하여 2와 3을 반환합니다.
