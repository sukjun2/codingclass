프로그래머스 1단계 : 없는 숫자 더하기

0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 numbers가 매개변수로 주어집니다. numbers에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.


※ 제한사항

1 ≤ numbers의 길이 ≤ 9
- 0 ≤ numbers의 모든 원소 ≤ 9
- numbers의 모든 원소는 서로 다릅니다.

※ 입출력 예

numbers answer
[1,2,3,4,6,7,8,0] 14
[5,8,4,0,6,7,9] 6 

function solution(numbers) {
    let answer = 0;
    let x = numbers.reduce((a, b) => a + b);
    answer = 45 - x;
    return answer;
}

x변수에 reduce() 메서드를 사용하여 numbers 매개변수의 배열 내부 숫자를 모두 합해줍니다. 그리고 0부터 9까지를 모두 더하면 45이므로, 45에서 합한 수를 빼주면 numbers에서 찾을 수 없는 0~9까지의 수를 합한 결과를 리턴하게됩니다.

※ 다른 사람의 풀이

function solution(numbers) {
    let answer = 0;

    for(let i = 0; i <= 9; i++) {
        if(!numbers.includes(i)) answer += i;
    }

    return answer;
}

includes()메서드를 사용하여 푼 방법입니다. for문에 i값의 조건을 설정하고, if문을 사용하여 numbers에 includes되지 않은 값을 answer변수에 합해주면 됩니다.
