프로그래머스 1단계 : 내적

길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어집니다. a와 b의 내적을 return 하도록 solution 함수를 완성해주세요.
이때, a와 b의 내적은 a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1] 입니다. (n은 a, b의 길이)


※ 제한사항

a, b의 길이는 1 이상 1,000 이하입니다.
a, b의 모든 수는 -1,000 이상 1,000 이하입니다.

※ 입출력 예

a b result
[1,2,3,4] [-3,-1,0,2] 3
[-1,0,1] [1,0,-1] -2 

function solution(a, b) {
    let answer = 0;
    for(i=0; i<a.length; i++){
        answer += a[i]*b[i];
    }
    return answer;
}

두 벡터의 각 성분끼리의 곱의 합산을 내적이라고 합니다.
for문을 사용해 a와 b의 인덱스 0부터 a의 길이 값 까지 곱한값을 누적하여 answer에 더해주면 끝입니다.

※ 다른 사람의 풀이

function solution(a, b) {
    return a.map((a,i)=> a*b[i]).reduce((acc,val)=>{
         return acc+val
     },0)
 }

a * b 배열 엘리먼트의 인덱스 위치 값을 각각 곱해주고 reduce문으로 모두 합해주면 끝이네요.
