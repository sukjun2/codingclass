프로그래머스 1단계 : 최대공약수와 최소공배수

두 수를 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수, solution을 완성해 보세요. 배열의 맨 앞에 최대공약수, 그다음 최소공배수를 넣어 반환하면 됩니다. 예를 들어 두 수 3, 12의 최대공약수는 3, 최소공배수는 12이므로 solution(3, 12)는 [3, 12]를 반환해야 합니다.


※ 제한사항

두 수는 1이상 1000000이하의 자연수입니다.

※ 입출력 예

n m return
3 12 [3, 12]
2 5 [1, 10] 

function solution(n, m) {
    let answer = [];
    let x = 0;
    for(let i = 1; i <= n; i++){
        if(n % i === 0 &&  m % i === 0){
            answer = [i];
            x = Math.max(m * n / i); 
        }
    }
    answer.push(x);
    return answer;
}

for문안에 조건문을 사용하여 n, m 두 수의 최대공약수를 먼저 찾아주고 최소공배수를 찾는 공식(a * b / 두 수의 최대공약수)을 사용하여 최소공배수를 찾아 answer에 push해주면 끝입니다.

※ 다른 사람의 풀이

function gcdlcm(a, b) {
    var r;
    for(var ab= a*b;r = a % b;a = b, b = r){}
    return [b, ab/b];
}
