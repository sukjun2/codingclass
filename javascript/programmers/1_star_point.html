프로그래머스 1단계 : 직사각형 별찍기

이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다. 별(*) 문자를 이용해 가로의 길이가 a, 세로의 길이가 b인 직사각형 형태를 출력해보세요.


※ 제한사항

n과 m은 각각 1000 이하인 자연수입니다.

※ 입출력 예

a b
5 3 

process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
    const n = data.split(" ");
    const a = Number(n[0]), b = Number(n[1]);
    
    for(i = 1; i <= b; i++){
        let star = '';
        for(j = 1; j <=a; j++){
            star += '*';
        }
    console.log(star);        
    }
});

예전에 배웠던 구구단 만들기와 비슷한 풀이입니다. for문을 사용해 세로줄인 b 실행 한번에 가로줄인 a가 횟수만큼 실행되도록 작성해주면 끝입니다. 특이하게 console로 답을 체크하네요.

※ 다른 사람의 풀이

process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
    const n = data.split(" ");
    const a = Number(n[0]), b = Number(n[1]);
    const row = '*'.repeat(a)
    for(let i =0; i < b; i++){
        console.log(row)
    }

});

row변수에 문자열 *을 repeat메서드로 a번만큼 반복시킵니다. 그리고 for문을 사용해 row변수를 b번 실행시키면 완성입니다.
