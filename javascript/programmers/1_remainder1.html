프로그래머스 1단계 : 나머지가 1이 되는 수 찾기

자연수 n이 매개변수로 주어집니다. n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 return 하도록 solution 함수를 완성해주세요. 답이 항상 존재함은 증명될 수 있습니다.


※ 제한사항

3 ≤ n ≤ 1,000,000

※ 입출력 예

n result
10 3
12 11 

function solution(n) {
    let answer = 0;
    for (let x = n - 1; x >= 2; x--) {
        if(n % x === 1) answer = x
    }
    return answer;
}

n값이 10, 12일 때 나머지가 1이 되도록하는 가장 작은 자연수 x는 10 % 3 == 1, 12 % 11 == 1 이므로, 3과 11이 출력되도록 for문을 작성하여 x--값을 적용하면 됩니다.
