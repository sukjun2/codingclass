프로그래머스 1단계 : 짝수와 홀수

정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.


※ 제한사항

num은 int 범위의 정수입니다.
0은 짝수입니다

※ 입출력 예

num return
3 "Odd"
4 "Even" 

function solution(num) {
    let answer = '';
    if(num % 2 == 0){
        answer = 'Even';
    } else {
        answer = 'Odd';
    }
    return answer;
}

num값을 2로 나눈 나머지값이 0이면 짝수가 되므로 Even을 출력하고, 아니라면 Odd를 출력하도록 if문을 사용하여 작성합니다.
