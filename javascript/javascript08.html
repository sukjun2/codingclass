  • 01. 숫자 메서드 : Number.Max_VALUE : 표현할 수 있는 제일 큰 양의 숫자 값을 나타냄
  • 02. 숫자 메서드 : Number.Min_VALUE : 표현할 수 있는 제일 작은 양의 숫자 값을 나타냄
  • 03. 숫자 메서드 : Number.MAX_SAFE_INTEGER : 안전한 최대 정수 값을 나타냄
  • 04. 숫자 메서드 : Number.MIN_SAFE_INTEGER : 안전한 최소 정수 값을 나타냄
  • 05. 숫자 메서드 : Number.NaN : Not-A-Number(숫자가 아님)를 나타냄
  • 06. 숫자 메서드 : Number.isFinite : 주어진 값이 유한수인지 판별
  • 07. 숫자 메서드 : Number.isInteger : 주어진 값이 정수인지 판별
  • 08. 숫자 메서드 : Number.isNaN : 주어진 값이 NaN인지 판별
  • 09. 숫자 메서드 : Number.prototype.toFixed : 숫자를 고정 소수점 표기법(fixed-point notation)으로 표시
  • 10. 숫자 메서드 : Number.prototype.toPrecision : Number 객체를 지정된 정밀도로 나타내는 문자열을 반환
  • 11. 숫자 메서드 : Number.prototype.toString : 숫자를 문자 혹은 진수로 변환


  • 01. 데이트 메서드 : new Date() : 아무런 인수를 전달하지 않으면 현재 날짜와 시간을 가지는 인스턴스를 리턴
  • 02. 데이트 메서드 : Date.now() : UTC 기준으로 1970년 1월 1일 0시 0분 0초부터 현재까지 경과된 밀리 초를 반환
  • 03. 데이트 메서드 : Date.parse() : 1970 년 1 월 1 일 00:00:00 UTC 이후의 밀리 초 수를 반환하거나 문자열이 인식되지 않거나 잘못된 날짜 값을 포함하는 경우 NaN을 반환
  • 04. 데이트 메서드 : Date.UTC() : Date 생성자와 비슷한 매개변수를 받지만, 모두 UTC로 취급, 1970년 1월 1일 00:00:00 UTC부터 매개변수가 나타내는 시간의 차이를 밀리초로 나타낸 수를 반환
  • 05. 데이트 메서드 : Date.prototype.getFullYear() : 주어진 날짜의 현지 시간 기준 연도를 반환
  • 06. 데이트 메서드 : Date.prototype.setFullYear() : 현지 시간에 따라 지정된 날짜의 전체 연도를 설정, 새로운 타임 스탬프를 반환
  • 07. 데이트 메서드 : Date.prototype.getMonth() : Date 객체의 월 값을 현지 시간에 맞춰 반환(월은 0부터 시작)
  • 08. 데이트 메서드 : Date.prototype.setMonth() : 현재 설정된 연도에 따라 지정된 날짜의 월을 설정
  • 09. 데이트 메서드 : Date.prototype.getDate() : 주어진 날짜의 현지 시간 기준 일을 반환
  • 10. 데이트 메서드 : Date.prototype.setDate() : 현재 설정된 월의 시작 부분을 기준으로 Date 객체의 날짜를 설정
  • 11. 데이트 메서드 : Date.prototype.getDay() : 주어진 날짜의 현지 시간 기준 요일을 반환(0은 일요일을 나타냄)
  • 12. 데이트 메서드 : Date.prototype.getHours() : 주어진 날짜의 현지 시간 기준 시를 반환
  • 13. 데이트 메서드 : Date.prototype.setHours() : 현지 시간에 따라 지정된 날짜의 시간을 설정하고 1970 년 1 월 1 일 00:00:00 UTC 이후 업데이트 된 Date에 의해 표시되는 시간 (밀리 초)을 반환
  • 14. 데이트 메서드 : Date.prototype.getMinutes() : Date 인스턴스의 분을 현지 시간 기준으로 반환
  • 15. 데이트 메서드 : Date.prototype.setMinutes() : 현지 시간에 따라 지정된 날짜의 분을 설정
  • 16. 데이트 메서드 : Date.prototype.getSeconds() : Date 객체의 초 값을 현지 시간에 맞춰 반환
  • 17. 데이트 메서드 : Date.prototype.setSeconds() : 현지 시간에 따라 지정된 날짜의 초를 설정
  • 18. 데이트 메서드 : Date.prototype.getMilliseconds() : Date 인스턴스의 밀리초를 현지 시간 기준으로 반환
  • 19. 데이트 메서드 : Date.prototype.setMilliseconds() : 현지 시간에 따라 지정된 날짜의 밀리 초를 설정
  • 20. 데이트 메서드 : Date.prototype.getTime() : 표준시에 따라 지정된 날짜의 시간에 해당하는 숫자 값을 반환
  • 21. 데이트 메서드 : Date.prototype.setTime() : Date 객체를 1970 년 1 월 1 일 00:00:00 UTC부터 밀리 초 단위로 나타내는 시간으로 설정
  • 22. 데이트 메서드 : Date.prototype.getTimezoneOffset() : 현재 로케일 (즉, 호스트 시스템 설정)에 대한 시간대 오프셋 (UTC)을 분 단위로 반환
  • 23. 데이트 메서드 : Date.prototype.toDateString() : 미국 영어로 사람이 읽을 수있는 형태로 Date 객체의 날짜 부분을 반환
  • 24. 데이트 메서드 : Date.prototype.toTimeString() : 현지 시간대에서 해석된 Date 객체 의 시간 부분을 영어로 반환
  • 25. 데이트 메서드 : Date.prototype.toISOString() : 단순화한 확장 ISO 형식(ISO 8601)의 문자열을 반환
  • 26. 데이트 메서드 : Date.prototype.toLocaleString() : 지정된 지역에서 표현하는 방식의 날짜를 문자열로 리턴
  • 27. 데이트 메서드 : Date.prototype.toLocaleTimeString() : 날짜의 시간 부분을 지역에 맞는 언어 포맷으로 반환
